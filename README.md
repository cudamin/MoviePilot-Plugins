# 空间清理器使用方法

使用时需在EMBY设置中的通知处配置Webhooks，Events需勾选播放。网址格式：

[http://MP容器ip地址:3001/api/v1/webhook?token=MP设置里的api令牌](http://<MP容器IP地址>:3001/api/v1/webhook?token=<MP设置里的API令牌>)
